# AiTrans 0.2.0

## 中文说明

### 变更摘要

- 发布 `0.2.0` 版本，将截图翻译交付方式从“结果面板”推进到“AI 对话浮窗”
- 新增悬浮入口图标，支持点击展开对话窗、右键进入托盘同类功能、拖拽调整停靠位置
- 对话窗改成锚点式浮窗，围绕悬浮入口弹出，不再独立拖拽和缩放
- 截图结果与文本输入统一进入本地会话线程，并支持恢复最近会话
- 将会话缓存下沉到主进程本地文件存储，支持“新聊天 / 清空记录”后确认删除本地记录
- 继续收紧对外界面，移除不应面向普通用户公开的执行信息与诊断模块

### 对使用者可见的影响

- 截图后的结果现在会像和 AI 对话一样出现在聊天线程中，而不是单次结果卡片
- 可以直接通过悬浮入口打开或收起对话窗口，入口感知更强
- 文本输入、截图翻译和继续追问现在都能落在同一条会话里
- 会话内容会在本地恢复；只有在用户明确新建会话或清空记录时才删除
- 界面公开层更精简，更接近 C 端产品而不是调试面板

### 安装或获取方式

- 安装版：运行 `desktop-screenshot-translate-0.2.0-x64.exe`
- 便携版：运行 `desktop-screenshot-translate-0.2.0-x64-portable.exe`
- 若要测试自动更新，请为已安装版配置：
  - `release.update_base_url`
  - 或 `AITRANS_UPDATE_BASE_URL`

### 已知限制

- 当前公开交付仍以 Windows 为主，macOS 尚未进入正式发布范围
- 当前未完成代码签名，部分 Windows 环境可能出现额外安全提示
- 若未提供可达的更新源，自动更新会保持禁用并回退为手动升级
- 若未提供可达的翻译端点与凭据，应用会退回本地 stub 翻译
- AI 对话浮窗的视觉、滚动跟随和输入区体验仍有继续打磨空间

## English Notes

### Summary

- Release `0.2.0` and move delivery from a single result panel toward an AI-style floating conversation window
- Add a floating anchor icon that supports click-to-toggle, right-click entry actions, and drag-to-reposition behavior
- Convert the chat window into an anchor-attached floating window instead of an independently draggable and resizable panel
- Unify screenshot results and typed input into one local conversation thread with last-session restore
- Move conversation persistence into main-process local file storage and require confirmation before deleting local history
- Continue simplifying the public-facing UI by removing execution and diagnostics sections that should not be exposed to end users

### User-visible impact

- Screenshot results now appear in a conversation thread, closer to an AI chat flow instead of a one-off result card
- The floating anchor provides a stronger desktop entry point for opening and collapsing the chat window
- Typed input, screenshot translation, and follow-up interaction now happen in the same conversation
- Conversation history is restored locally and only cleared after an explicit user confirmation
- The visible UI is more product-focused and less like an internal debugging surface

### Install / Download

- Installer: run `desktop-screenshot-translate-0.2.0-x64.exe`
- Portable: run `desktop-screenshot-translate-0.2.0-x64-portable.exe`
- To test auto update, configure:
  - `release.update_base_url`
  - or `AITRANS_UPDATE_BASE_URL`

### Known limitations

- Public delivery still targets Windows first; macOS is not part of the formal release scope yet
- Code signing is not finished yet, so some Windows environments may show additional security prompts
- If no reachable update feed is configured, auto update stays disabled and the app falls back to manual upgrade
- If no reachable translation endpoint and credential are provided, the app falls back to local stub translation
- The AI conversation floating window still needs more refinement around visual polish, bottom-follow scrolling, and composer behavior
