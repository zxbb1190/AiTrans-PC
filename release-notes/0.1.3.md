# AiTrans 0.1.3

## 中文说明

### 变更摘要

- 发布 `0.1.3` 版本，继续收敛 Windows 端结果面板、托盘入口和首次配置界面
- 将结果面板重排为“轻量状态栏 + 原文译文双栏 + 底部动作栏 + 默认折叠详情”
- 收紧托盘顶级入口，移除不应直接暴露给普通用户的结果面板与配置路径动作
- 收紧首次配置主路径，支持“保存并开始截图”，并将高级排障降为折叠区域
- 修复截图完成后 overlay 可能再次出现并卡住的问题
- 为无边框结果窗口补上可拖动区域

### 对使用者可见的影响

- 结果面板首屏现在更聚焦原文、译文和动作，不再被大块装饰和诊断信息抢占
- 托盘菜单现在更短、更偏产品入口语义
- 首次配置窗口更接近正式使用路径，普通用户不需要先理解配置文件
- 截图完成后更不容易再次卡回遮罩层
- 结果窗口现在可以拖动调整位置

### 安装或获取方式

- 安装版：运行 `desktop-screenshot-translate-0.1.3-x64.exe`
- 便携版：运行 `desktop-screenshot-translate-0.1.3-x64-portable.exe`
- 若要测试自动更新，请为已安装版配置：
  - `release.update_base_url`
  - 或 `AITRANS_UPDATE_BASE_URL`

### 已知限制

- 当前公开交付仍以 Windows 为主，macOS 尚未进入正式发布范围
- 当前未完成代码签名，部分 Windows 环境可能出现额外安全提示
- 若未提供可达的更新源，自动更新会保持禁用并回退为手动升级
- 若未提供可达的翻译端点与凭据，应用会退回本地 stub 翻译
- 结果面板、托盘与首次配置界面的视觉和交互仍有继续打磨空间

## English Notes

### Summary

- Release `0.1.3` to continue refining the Windows result panel, tray entry, and first-run setup experience
- Restructure the result panel into a lightweight status bar, dual-column source/translation body, bottom action row, and collapsed details section
- Tighten the tray top-level entries by removing end-user-facing access to the result stub and config-path actions
- Tighten the first-run setup main path with a “save and start capture” flow while moving troubleshooting actions into a collapsed section
- Fix the issue where the capture overlay could reappear and get stuck after a capture was submitted
- Add a draggable region to the frameless result window

### User-visible impact

- The result panel now prioritizes source text, translated text, and actions on the first screen
- The tray menu is now shorter and closer to the intended product entry flow
- The first-run setup window is closer to a real user path and no longer leads with config-file concepts
- The capture overlay is less likely to reappear and block interaction after a capture completes
- The result window can now be dragged to a new position

### Install / Download

- Installer: run `desktop-screenshot-translate-0.1.3-x64.exe`
- Portable: run `desktop-screenshot-translate-0.1.3-x64-portable.exe`
- To test auto update, configure:
  - `release.update_base_url`
  - or `AITRANS_UPDATE_BASE_URL`

### Known limitations

- Public delivery still targets Windows first; macOS is not part of the formal release scope yet
- Code signing is not finished yet, so some Windows environments may show additional security prompts
- If no reachable update feed is configured, auto update stays disabled and the app falls back to manual upgrade
- If no reachable translation endpoint and credential are provided, the app falls back to local stub translation
- The result panel, tray, and setup UI still need more product-level refinement
