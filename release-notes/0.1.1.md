# AiTrans 0.1.1

## 中文说明

### 变更摘要

- 首次发布 `AiTrans` Windows 版桌面截图翻译工具
- 提供托盘入口、全局快捷键、区域截图与整屏截图
- 提供本地 OCR 与 OpenAI 兼容翻译端点接入
- 提供首次配置界面，可直接填写翻译端点与凭据
- 提供 `NSIS` 安装包与 `portable` 便携版
- 提供 Windows 安装版应用内自动更新接入

### 对使用者可见的影响

- 安装后可从托盘或快捷键启动截图翻译
- 首次启动若未配置翻译端点，会自动打开配置窗口
- OCR 运行时已随应用分发，不要求终端用户手工安装 `tesseract`
- Windows 安装版可在配置更新源后使用应用内自动更新
- `portable` 便携版仍通过重新下载新包完成升级

### 安装或获取方式

- 安装版：运行 `desktop-screenshot-translate-0.1.1-x64.exe`
- 便携版：运行 `desktop-screenshot-translate-0.1.1-x64-portable.exe`
- 首次启动后，可在应用界面中填写：
  - `translation.base_url`
  - `translation.api_key`

### 已知限制

- 当前公开交付以 Windows 为主，macOS 仍未进入正式发布范围
- 当前未完成代码签名，部分 Windows 环境可能出现额外安全提示
- 若未提供可达的翻译端点与凭据，应用会退回本地 stub 翻译
- 若未提供可达的更新源，自动更新会保持禁用并回退为手动升级

## English Notes

### Summary

- First public Windows release of `AiTrans` desktop screenshot translation
- Includes tray entry, global shortcut, region capture, and fullscreen capture
- Includes bundled local OCR and OpenAI-compatible translation endpoint support
- Includes a first-run setup window for translation endpoint and credential entry
- Ships both `NSIS` installer and `portable` package
- Includes in-app auto update integration for the Windows installed build

### User-visible impact

- Screenshot translation can be started from the tray or the global shortcut
- If no translation endpoint is configured, the app opens the setup window automatically on first launch
- OCR runtime is bundled with the app; end users are not expected to install `tesseract` manually
- The Windows installed build can use in-app auto update once a reachable update feed is configured
- The portable build still upgrades by downloading a new package manually

### Install / Download

- Installer: run `desktop-screenshot-translate-0.1.1-x64.exe`
- Portable: run `desktop-screenshot-translate-0.1.1-x64-portable.exe`
- After first launch, configure:
  - `translation.base_url`
  - `translation.api_key`

### Known limitations

- Public delivery currently targets Windows first; macOS is not part of the formal release scope yet
- Code signing is not finished yet, so some Windows environments may show additional security prompts
- If no reachable translation endpoint and credential are provided, the app falls back to local stub translation
- If no reachable update feed is configured, auto update stays disabled and the app falls back to manual upgrade
