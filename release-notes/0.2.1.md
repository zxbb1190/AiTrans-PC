# AiTrans 0.2.1

## 中文说明
- 修复截图完成后对话浮窗偶发不自动显示的问题。
- 优化清晰单行英文文本的 OCR 识别策略，降低中文模型抢跑带来的误识别。
- 调整对话浮窗的显示保护时序，改善未固定状态下的结果交付稳定性。

## English Notes
- Fixed an intermittent issue where the chat window did not reopen after capture completed.
- Improved OCR for clear single-line English text to reduce misrecognition caused by non-English-first attempts.
- Refined the chat window show timing to make result delivery more reliable when the window is not pinned.
