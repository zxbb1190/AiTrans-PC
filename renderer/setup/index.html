安装包已经内置 OCR。首次启动只需要把翻译端点配置补齐，之后就可以直接通过托盘或快捷键开始截图翻译。

你要做什么
  • 打开配置文件，把 `translation.base_url` 指向你的 OpenAI 官方或兼容端点。
  • 填入 `translation.api_key`。如果你的局域网端点允许无密钥，可留空。
  • 保存后回到托盘，重新截图即可生效，不必重新安装。
补充说明
  • 默认会自动生成 `runtime-overrides.json` 模板。
  • 如果你使用 OpenAI 官方端点，必须提供有效 API Key。
  • 如果你使用局域网兼容端点，只要端点可达并返回兼容结果即可。
