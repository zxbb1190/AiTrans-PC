# AiTrans 截图翻译

[English](./README.md)

AiTrans 是一个面向 Windows 的桌面截图翻译与 AI 对话辅助应用。
这份 README 优先给使用者看，重点说明安装、首次配置、日常使用和更新方式。

## AiTrans 可以做什么

- 截图后识别原文
- 通过 OpenAI 官方 API、DeepSeek API、智谱 AI API，或其他 OpenAI-compatible 服务完成翻译
- 在轻量 AI 对话浮窗中继续追问或继续翻译
- 将当前会话保存在本地，直到你选择新聊天或清空记录

## 安装方式

你可以选择以下任一安装包：

- 安装版：`desktop-screenshot-translate-<version>-x64.exe`
- 便携版：`desktop-screenshot-translate-<version>-x64-portable.exe`

普通用户推荐优先使用安装版。

## 首次启动

首次启动后，AiTrans 会：

- 在桌面显示悬浮图标
- 自动生成本地运行时配置文件
- 在翻译配置缺失时自动打开“设置与连接”窗口

首次至少需要配置：

- 翻译服务
- 模型名称
- `base_url`
- `api_key`

## 当前支持的翻译服务

AiTrans 当前支持：

- OpenAI 官方 API
- DeepSeek API
- 智谱 AI API
- 其他提供 OpenAI-compatible 接口的服务

这并不等于“所有模型厂商的原生 API 都能直接接入”。
如果某家厂商只提供自己的私有协议，而不是 OpenAI-compatible 接口，那么仍然需要单独适配。

## 建议先检查的配置

在“设置与连接”里，建议先确认这些内容：

- 翻译服务预设
- 模型名称
- 翻译服务地址：`base_url`
- 翻译服务密钥：`api_key`
- 源语言：自动 / 中文 / 英文 / 日文
- 截图快捷键
- 发送消息快捷键

## 如何使用

### 截图翻译

- 点击悬浮图标，或使用截图快捷键
- 拖动鼠标框选截图区域
- 多屏环境下，把鼠标移到另一块屏幕即可切换当前截图目标屏
- 截图完成后，对话浮窗会自动弹出并显示结果

### 对话浮窗

- 一次截图会作为一条用户消息进入当前会话
- AI 回复会包含原文和译文
- 你也可以继续在底部输入框里追问或直接要求翻译
- 未固定时，浮窗失焦会自动收起
- 固定后，浮窗会保持显示

### 悬浮图标

- 左键：展开或收起对话浮窗
- 拖动：移动悬浮图标位置
- 右键：打开菜单，可执行新聊天、清空记录、设置与连接、检查更新等操作

## 本地记录

- 当前对话会保存在本地
- 点击“新聊天”或“清空记录”时，应用会先确认，再删除当前会话缓存

## OCR 与翻译说明

- OCR 现在会优先尝试本地 PaddleOCR 运行时；若未配置或失败，再回退到应用随包分发的 Tesseract。
- 翻译支持 OpenAI 官方端点，以及 DeepSeek、智谱 AI 和其他 OpenAI-compatible 端点
- 如果清晰的英文单行标题识别不准，建议配置本地 PaddleOCR 运行时；若清晰的英文单行标题识别不准，也建议先把源语言切到“英文”后再截图

## 更新说明

- 安装版支持检查更新
- 便携版不支持自动更新，建议手动替换到新版
- 如果已经配置更新源，可以从右键菜单触发检查更新
